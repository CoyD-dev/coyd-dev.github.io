Touch Target Spacing

Minimum spacing between interactive elements:

  • Horizontal: 8px minimum, 12-16px recommended
  • Vertical: 8px minimum, 12-16px recommended
  • Critical actions: 16px+ spacing for safety

⚠️ Common Mobile UX Issues to Avoid

  • ❌ Buttons smaller than 44×44px
  • ❌ Text links without adequate height/padding
  • ❌ Close proximity between destructive and safe actions
  • ❌ Input fields with font-size < 16px (causes iOS zoom)

⚡ Mobile Performance Optimizations:

  • ✅ Reduced blur (expensive on mobile GPUs)
  • ✅ Simplified shadows (better battery life)
  • ✅ Use transform/opacity for animations (GPU-accelerated)
  • ✅ Lazy load images below the fold
  • ✅ Avoid backdrop-filter on scrolling elements

🎯 Target Performance Metrics

  • Lighthouse Mobile Score: ≥ 90
  • First Contentful Paint: < 2s on 3G
  • Time to Interactive: < 3.5s on 3G
  • Scroll Performance: Consistent 60fps
